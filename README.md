я создал телефон из ардуино. Меня можно увидеть в рутубе под названием neiro arduino. Я этот телефон назвал аруфон. Он создан на базе JC4827W543. У вас должно быть точно такие же настройки как  у меня. Сначало вы скачайте файлы для esp32, я выбрал версию 2.0.10. после нажимайете на саму esp32 и там у вас есть все платы. Выбераем esp32s3 dev module и выставляем такие настройки как на этой фотографии.
